### 关于本项目
#### 1.demo文件夹
存放一些示例代码
#### 2.expressLearn文件夹
存放express学习笔记及学习示例代码
#### 3.mongodb文件夹
存放mongodb学习笔记及学习示例代码
#### 4.note文件夹
存放一些笔记
#### 5.ohShit文件夹
存放一些踩过的坑及一些未解决的坑
#### 6.question文件夹
存放整理的面试题，word格式
#### 7.mdGrammar.md
存放Markdown常用语法

#### 解释
笔记文件一般命名为note.md，查看markdown文件，建议使用vscode，安装插件Markdown Preview Enhanced预览，因为流程图等一些markdown语法，github上直接查看不显示
