## React17.x 项目，ts配置文件报错：Argument for '--jsx' option must be: 'preserve', 'react-native', 'react'，tsx文件报错：Cannot use JSX unless the '--jsx' flag is provided

1. 打开一个tsx文件
2. Ctrl+Shift+P，输入“select typescript version”，选择“使用工作区版本”
